Telegram slavarikk Bot — учи слова с удовольствием!
Интерактивный бот для изучения иностранных слов. Добавляй свои слова, отмечай выученные и проходи увлекательные квизы!

Что умеет бот
Запускай квиз командой /quiz и отвечай на вопросы, выбирая правильный перевод из четырёх вариантов. После каждого ответа сразу видишь — правильно или нет, а в конце получаешь свой результат.

Весь словарь управляется простыми командами через сообщения. Можно добавлять новые слова, проверять перевод, отмечать выученные и удалять ненужные.

Как пользоваться
Всё строится на простых текстовых командах. Чтобы добавить слово, просто напиши его и перевод через двоеточие:
привет: hello

Проверить слово можно знаком вопроса перед ним:
?привет

Если слово уже выучил — поставь плюсик:
+привет

Ну а чтобы удалить из словаря — минус:
-привет

Для начала квиза просто отправь /quiz. Бот выберет пять случайных слов и покажет вопрос с кнопками. После ответа увидишь результат, а затем сможешь перейти к следующему вопросу.

Как это устроено внутри
Бот хранит словарь каждого пользователя в памяти. Для каждого слова запоминается перевод и статус — выучено или нет.

Когда запускается квиз, бот собирает пять случайных слов, для каждого подбирает три неправильных варианта из того же словаря и показывает вопрос. После ответа всё обновляется прямо в том же сообщении, а в конце квиз удаляется из памяти.

Что нужно для запуска
Понадобится Python версии 3.7 или новее и библиотека pyTelegramBotAPI. Установить её можно одной командой:

pip install pyTelegramBotAPI

Создай файл config.py, положи туда токен своего бота, который получил от @BotFather:

TOKEN = "твой_токен_сюда"

После этого можно запускать:

python bot.py

Немного примеров
Добавляем слова:
кот: cat
собака: dog
машина: car

Проверяем:
?кот
Бот отвечает: кот: cat ✗

Отмечаем выученное:
+кот
Бот: Выучено!

Запускаем квиз:
/quiz
Бот показывает первый вопрос с вариантами ответов в виде кнопок.
